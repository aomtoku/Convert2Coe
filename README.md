Convert2Coe
===========

This program converts PPM to CORE(*.core)

the format is
   - A "Magic Number"  -- P6 or P3
        P6:
	p3:
   - White Space.(0x0a)
   - A Width
   - White Space.(0x0a)
   - A Height.
   - White Space.(0x0a)
   - The Maximum Color - Component Value
   - White Space.
